component-controls allows full customization of the user interface - from theming, SEO tags to toolbars, sidebars and footer. Most of the user-interface customizations can be set up in your .`config/runtime.js` file.

## Installing a theme

You can use any [theme-ui](https://theme-ui.com) theme as a starting point for customizing the user interface.

<Story name="select-theme" controls={{ theme: { type: ControlTypes.OPTIONS, options: ['none', ...Object.keys(themes)], value: 'none' }}}>
  {({ theme: themeName }) => {
    const store = useStore();
    const [globalTheme, setGlobalTheme] = useThemeState();
    const theme = themes[themeName] || store.config.theme;
    useEffect(() => {
      if (globalTheme !== theme) {
        setGlobalTheme(theme)
      }
    }, [theme]);  
    return (
        <Source title=".config/runtime.js">
{`${themeName !== 'none' ? `import { ${themeName} } from '@theme-ui/presets';` : ''}
export default {
  theme: ${themeName !== 'none' ? themeName : '{}'},
}`}
      </Source>
    );  
  }}
</Story>

## Toolbar menu items

You can add buttons and links to the toolbar - both on the laft-hand side and the right-hand side.

export const ButtonControls = (label) => ({
  visible: { type: ControlTypes.BOOLEAN, value: true },
  side: { type: ControlTypes.OPTIONS,  options: ['left', 'right'], value: 'left', display: 'inline-check' },
  text: { type: ControlTypes.TEXT, value: label },
});
